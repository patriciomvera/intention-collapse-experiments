# Documentation

This folder contains documentation and reference materials for the Intention Collapse experiments.

## Structure

```
docs/
├── paper/
│   └── intention_collapse.pdf    # Original paper (add your copy here)
├── notes/                        # Research notes (optional)
└── references/                   # Additional references (optional)
```

## Paper

Place your copy of the paper in `docs/paper/`:

- **Filename**: `intention_collapse.pdf` (recommended)
- **Title**: "Intention Collapse: A Unified Framework for Understanding Reasoning in Language Models"

## Notes

You can add research notes, experiment logs, or analysis documents here.

## References

Additional papers or resources that informed this work can be stored in `references/`.
